# Light & Dark Mode

Light and Dark Mode is a study in Javascript and how to dynamically create a light and dark mode setting for web applicaitons and web sites. This page is not fully responsive yet.

## Features

- Light/dark mode toggle
- Session Storage
- Responsive (WIP)
